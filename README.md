# Movie-Recommendation-System
A Content-Based Movie Recommendation System built using Python and Machine Learning. The project recommends similar movies based on genres and user-generated tags using TF-IDF Vectorization and Cosine Similarity. It also includes data preprocessing, exploratory data analysis (EDA), feature engineering, and model evaluation.
# 🎬 Movie Recommendation System

A Content-Based Movie Recommendation System developed using Python and Machine Learning. This project recommends similar movies based on genres and user-generated tags by applying TF-IDF Vectorization and Cosine Similarity.

---

## 🎯 Objectives

- Build a Content-Based Movie Recommendation System.
- Perform data cleaning and preprocessing.
- Analyze movie data using Exploratory Data Analysis (EDA).
- Generate movie recommendations using Machine Learning.
- Evaluate the recommendation results.

---

## 📂 Dataset

The project uses the MovieLens dataset consisting of:

- **movies.csv** – Movie titles and genres
- **ratings.csv** – User ratings
- **tags.csv** – User-generated tags
